# [`tutrhq`](http://tutrhq.com) sources

### Description
Welcome to the source code of tutr! 

### Installation

``` 
$ sudo apt-get install npm
$ sudo apt-get install nodejs
$ npm install express
$ npm install formidable

```

### How to start `tutr`
```;
$ cd app
$ npm install
$ npm start
  
```
You should have a running instance at `http://localhost:3000`
